# YTCreate V2

YTCreate V2 is a greenfield, deterministic-first video pipeline.

Phase 0 establishes only the runnable skeleton:

- FastAPI backend with `GET /health`
- React/Vite frontend shell
- Remotion renderer scaffold
- ownership documentation
- smoke tests

No persistence, pipeline orchestration, engines, validators, artifact store, or rendering logic is implemented in Phase 0.
